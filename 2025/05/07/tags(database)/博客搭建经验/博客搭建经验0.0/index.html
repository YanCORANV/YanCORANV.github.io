博客搭建经验（前言）

Created2025-05-07|Updated2025-07-09
|Word Count:1.1k|Reading Time:3mins|Post Views:

这里会有一些关于搭建博客的心得，如果对于搭建一个像笔者这样的个人博客欢迎阅读~~

搭建博客需要什么能力

搭建个人博客听起来好像很难，需要前端开发的经验和Javascript的能力，但感谢这个开源的世界，完全没有写Javascript的经验的笔者也能搭起来（笑），以下是一些笔者个人认为搭建博客需要的能力

信息检索

之所以把这个放在第一条，是因为在搭博客的过程中感受到网上信息很多很多，如何用最短的时间找到合适自己的教程很重要，因此，学会高效的信息检索或许是搭建博客的第一步。以下是笔者个人检索信息的一些方式，供参考：

  • 查阅官方文档
  • b站搜索关键词（个人认为能在B站具有一定长度和播放量的视频相对来说会比某些平台上的不知名帖子高质量）
  • 在相关视频的b站评论区看大家的经验分享
  • 关注B站相关up主的个人博客，关注他们的友链
  • 在CSDN/Stackoverflow上检索
  • 在小红书/小绿书上检索
  • 问AI

基础工具使用

笔者所言基础工具包括：

  • 对于git bash的使用，换句话说是对于cmd的使用
  • 对于github的使用
  • 对于git的使用
  • 对于VSCode等带高亮的文本编辑器的使用（主要用于编辑一些yml文件）（突然想到把VSCode定义为文本编辑器有点过于不严谨，这里叠甲，大家意会就好）
    这些其实都不难，可以即学即用，并且笔者相信对于以后会有很大的帮助~

搭建博客的收获

  1. 拥有了一个可以自己畅所欲言的地方，很有成就感(●’◡’●)
  2. 在搭博客过程中不断遇到各种问题，真的是各种问题（笔者因为一些问题连续两天搞到凌晨四点半）。但是为了解决问题自己去检索信息，调试的过程，真的让笔者提升了自己解决问题的能力
  3. 对于git bash的使用更熟悉了，打开终端页面不再是不知所措，尽管平时在学校也会用（笑）
  4. 写markdown文件更顺手了
  5. 因为投入了很多的时间成本，所以会Push自己多写一些东西，笔者认为学习过程中有所输出也是很重要的

后记

  1. 博客的未来：目前有如下几个打算
    1. 同步一些笔者在学校的课程笔记，全部开源至github并在个人网站提供访问链接，供大家自由使用，不过大概率是提供PDF版本，因为md的插图功能偶尔会导致图片不可见的问题（这也是导致笔者某天熬夜到四点半的原因）
    2. 分享生活的一个渠道，包括但不限于在每日碎碎念栏目说一些小废话，以及马上开设的“每周计划”栏目，用于笔者把自己的计划公开，督促自己。
    3. 分享一些资源，例如笔者在网站栏目分享的一些网站，都是笔者自己，或者是被优秀的前辈们推荐给大家的，算是为自己做一个信息整理以及帮助大家打破信息差
    4. 笔者自己的资料库，说来惭愧，笔者一直有建设自己的资料库的打算，之前尝试过使用Notion但是也无疾而终了。希望未来可以在个人资料库同步如下内容
      1. 书籍推荐
      2. 电影推荐
      3. 音乐推荐
      4. 喜欢的句子摘录
    5. 干货分享，例如说笔者上过的觉得很好的课程，会连同课程+笔记一起放在网站上并在Github开源，然后有一些自己的心得（例如obsidian使用以及心态等等觉得有价值的东西都会分享在这）
  2. 应该还会专门开一个帖子写对于博客的展望（？）主要是当作对于建设博客的备忘录以及立flag
  3. 欢迎贡献：欢迎大家通过各种方式来联系笔者，提出建议/批评，互加友链，大家一起无限进步ヽ(￣ω￣(￣ω￣〃)ゝ
